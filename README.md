# SaaS Management System

## 🚀 Project Overview
This is a SaaS Management System built using MERN stack.

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## 📁 Project Structure
- backend/
- frontend/

## ⚙️ Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Create `.env` file
4. Run server

## 📌 Features (Planned)
- User Authentication
- Role-based Access Control
- Subscription Management
